# Financial-Risk-Analysis
Analyzed financial loan data using Python to identify key factors influencing loan default risk. Performed data cleaning, exploratory analysis, and risk segmentation based on credit scores. Built an interactive dashboard in Microsoft Power BI to visualize default trends, loan purpose risk, and borrower insights for data-driven decision making.

Project Title:
Financial Risk Analysis

Tools Used:
Python
Pandas
Matplotlib
Seaborn
Microsoft Power BI

Project Description:

This project analyzes loan default risk using customer financial data. The analysis identifies key factors influencing loan defaults such as credit score, income level, loan amount, and loan purpose. A Power BI dashboard was built to visualize risk patterns and support data-driven decision making.

Key Insights (Add This Section)
Findings:
* Overall loan default rate ≈ 22%
* High-risk customers default ~28%
* Business loans show the highest default rate
* Lower credit scores strongly correlate with higher default probability
